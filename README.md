# Claude ABAP PO Demo
Purchase Order list report generated by Claude Code and deployed via GitHub and abapGit.

## Report: ZCLABAP_PO_REPORT
Reads from EKKO (PO Header) joined with EKPO (PO Items). Displays in SAP ALV grid.

## Selection Parameters
- SO_EBELN — PO Number (optional)
- SO_BUKRS — Company Code (optional)
- SO_BEDAT — PO Date range (optional)
- SO_BSART — Document Type (optional)

## How to run
Transaction SE38 → enter ZCLABAP_PO_REPORT → F8
