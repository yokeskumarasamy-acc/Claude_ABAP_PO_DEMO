# Claude ABAP PO Demo
Purchase Order list report generated by Claude Code and deployed via GitHub and abapGit.

## Class: ZCLABAP_PO_REPORT
BTP ABAP Environment compatible. Implements `if_oo_adt_classrun`.
Reads from EKKO (PO Header) joined with EKPO (PO Items). Outputs to ADT console.

> Note: Classic REPORT programs (PROG) are not supported in SAP BTP ABAP Environment.
> This class uses `if_oo_adt_classrun` as the BTP-compatible equivalent.

## How to run
1. Open the class `ZCLABAP_PO_REPORT` in ADT (Eclipse)
2. Press **F9** (Run as ABAP Application Console)
3. Results appear in the ABAP Console view

## Fields displayed
- PO Number, Item, Company Code, Document Type, Vendor
- PO Date, Purchasing Org, Purchasing Group, Currency
- Material, Description, Quantity, UoM, Net Price, Net Value, Plant
